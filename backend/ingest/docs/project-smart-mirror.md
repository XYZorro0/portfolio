# AI-Powered Smart Mirror

Ongoing personal computer-vision project — this is what Niket is currently building. Tech: MediaPipe, computer vision, Python, Kinect, multi-GPU, local AI inference.

Niket is developing a smart mirror that uses MediaPipe for hand tracking and pose detection, with Kinect depth sensing for person recognition and auto-wake functionality — the mirror wakes up when it recognizes someone standing in front of it.

He is architecting a multi-GPU pipeline (a GTX 1070 plus a GTX 1060) that dedicates separate graphics cards to real-time computer vision processing and display rendering.

Planned features include photo-to-3D garment conversion and weather-aware outfit recommendations, all running on local AI inference rather than cloud services.
