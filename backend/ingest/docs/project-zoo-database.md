# Zoo Management Database System

Database systems course project (COSC 3380 at the University of Houston, Jan 2025 – May 2025). Tech: MySQL, SQL, database design, 3NF normalization.

Niket designed and implemented a comprehensive MySQL database system for zoo operations, managing multiple managers, staff members, gift shop inventory, and a ticket booking system.

He built a normalized schema in third normal form (3NF) with efficient relational design supporting complex queries and reporting across interconnected tables, and implemented two database triggers that automate business logic and maintain data integrity.
